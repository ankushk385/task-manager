# Task Management

This project contains a React frontend and a Node.js + MongoDB backend for a Task Management API.

- Backend: `backend/` — Express, Mongoose, JWT authentication
- Frontend: `frontend/` — React (vanilla create-react-app style), simple auth & tasks UI

Steps:

1. Setup backend: see `backend/README.md`
2. Setup frontend:
   - `cd frontend`
   - `npm install`
   - `npm start`

I built a minimal, working scaffold that meets the requirements: authentication, protected routes,
filtering, sorting, pagination, and validation.
